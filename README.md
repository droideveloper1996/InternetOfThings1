# Dex IoT App

App Allows You to control Devices remotely,has following features
 - GeoFencing Support
 - Notification Service
 - Scheduling Activity
 - Power Consumption
 - Data Logging to cloud
 - Each Device has unique key that need to be registerd with app
 
 Supported Hardware: ESP8266, ESP32
